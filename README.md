# User Connexion
Plug In personalisé de gestion de utilisateur indépendant aux fonction WP.

## Description
User Connexion est un plug in de gestion de compte utilisateur relié à sa propre database wp_, il permet de :
* Créer une base de donnée wp_ pour la gestion des utilisateurs externes.
* Gerer la connexion, inscription des utilisateurs.
* PHP/AJAX javascript pour la fluidité et eviter tout rafraichissement de pages.
* Redirection vers userpage si utilisateur connecté.

## Outils
* **HTML5** : Template du formulaire.
* **PHP** : Classes, Formulaires, gestion pont avec AJAX.
* **JavaScript** : Gestion du bouton de connexion, modal, AJAX lié au PHP.
